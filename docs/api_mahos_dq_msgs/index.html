mahos_dq.msgs#

The mahos_dq.msgs are message type definitions for mahos_dq.

confocal_msgs.Image([params])

Confocal image frame and acquisition state snapshot.

confocal_msgs.Trace([size, channels, _complex])

Multi-channel confocal trace buffer with timestamped samples.

odmr_msgs.ODMRData([params, label])

Data type for ODMR measurement.

podmr_msgs.PODMRData([params, label])

Pulse ODMR data container with TDC-derived raw traces.

spodmr_msgs.SPODMRData([params, label])

Slow-detector pulse ODMR data container.

qdyne_msgs.QdyneData([params, label])

Qdyne measurement data with time-domain and FFT representations.

iodmr_msgs.IODMRData([params])

Imaging ODMR data carrying cumulative and per-sweep image buffers.

hbt_msgs.HBTData([params])

HBT measurement data with raw and normalized correlation traces.

spectroscopy_msgs.SpectroscopyData([params])

Spectroscopy dataset with wavelength axis and stacked acquisitions.
